# 社員資料登入系統

這個系統用於紀錄和管理社員資料，功能包括新增社員、檢索社員資料、賦予職稱、列印所有社員資訊等，適合用於小型社團或團體的成員管理。

## 功能

1. **register** - 新增社員資料，包括名字、科系和學號。若重複輸入相同成員，會根據次數自動提升等級。
2. **search** - 根據不同條件查詢社員資訊，可以依名字、科系、學號、等級或職稱進行查詢。
3. **entitle** - 指定某個社員的職稱，若該社員已擁有「社長」職稱則無法重新指定。
4. **check** - 列印所有社員的資料，方便快速檢視社團成員狀態。
5. **help** - 顯示可用指令列表和使用說明。
6. **exit** - 結束程式。

## 錯誤處理

- 如果輸入的指令不符合格式或功能不存在，會顯示提示訊息。
- 預設情況下，輸入會按照正確格式提供，因此不需特別檢查錯誤。
